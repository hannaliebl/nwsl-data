About

Pro Women's Soccer in America

If you're reading this you probably already know about the National Women's Soccer League, but on the off chance you don't, please check their site and look into supporting your local team. The league boasts some of the best national and international talent.

Data and Technology

The data from this site comes from the NWSL Stats page, updated as of the end of the 2014 season. I converted it to JSON using Mr. Data Converter and encoded other additional information (like a team key per player) into the data sets by hand. If you believe there is an error with the data, please send me a message or a Tweet.

The site was built using Angular JavaScript as a framework. The charts and graphs are built using D3. The web server is Express. The development environment used Gulp and SASS. I made this as a learning project and to express my love for women's soccer. Take a look at the code on Github.
